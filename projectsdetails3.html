Website liliStore

Website presensi QR code ini merupakan sistem berbasis web yang dikembangkan menggunakan framework CodeIgniter dan menggunakan database lokal sebagai penyimpanan data. Website ini dirancang untuk memudahkan proses kehadiran baik bagi siswa maupun guru di lingkungan sekolah. Proses presensi dilakukan dengan cara memindai kode QR yang telah disediakan, sehingga lebih praktis dan efisien dibandingkan metode manual. Dalam sistem ini, admin memiliki peran sebagai pengelola utama yang bertanggung jawab untuk mengatur data pengguna, jadwal, serta monitoring data kehadiran. Dengan pendekatan ini, kehadiran dapat tercatat secara otomatis, akurat, dan terdokumentasi dengan baik.
